## Component Name : Mmi2x2
Author : Chung-Yu Hsu

- 50:50 2×2 MMI splitter
- Loss is ≤0.15±0.01 dB for 1530-1565 nm.
- Ref: https://doi.org/10.1109/JPHOT.2012.2230320
